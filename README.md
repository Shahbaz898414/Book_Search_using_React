# Book Search App

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Shahbaz898414/Book_Search_using_React.git
   cd book-search-app

2. Install dependencies:

   ```bash
   npm install

3. Start the development server:
  ```bash
  npm start


4. Open your browser
  navigate to http://localhost:3000 to see the application.
